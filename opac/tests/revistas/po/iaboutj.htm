Basic information

Polímeros is a quarterly publication of the Associação Brasileira de Polímeros - ABPol (Brazilian Polymer Association), which publishes Review Articles, Original Articles and Short Communications, disclosing advances in the knowledge of Polymer Science and Technology.

In 2018, the number of issues has been raised to five per year, with the following periodicity: 1° Issue (Jan, Feb, Mar), 2° Issue (Apr, May), 3° Issue (Jun, Jul), 4° Issue (Aug, Set) e 5° Issue (Oct, Nov, Dec).

The full name of this journal is “Polímeros: Ciência e Tecnologia”, which is to be used in any publication including references (bibliographies), footnotes, citation and quotations.
